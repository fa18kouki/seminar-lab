# 機械学習演習

このリポジトリは機械学習の基礎からディープラーニングまでを学ぶための演習課題を含んでいます。

## 概要

このコースでは以下の内容を学習します：

1. Python基礎
2. PyTorch入門
3. ニューラルネットワークの基礎
4. 画像認識タスク

## 演習内容

### 1. Python入門 (`1_intro.ipynb`)
- Python基本文法
- 数値計算
- リスト操作
- グラフ描画

### 2. PyTorch基礎 (`3_pytorch_tutorial.ipynb`)
- テンソル操作
- 自動微分（autograd）
- 基本的な計算グラフ

### 3. AND関数の学習 (`4_ANDfunction.ipynb`)
- ニューラルネットワークの基礎
- 順伝播と逆伝播
- 最適化手法

### 4. MNIST数字認識 (`5_mnist.ipynb`)
- 畳み込みニューラルネットワーク
- 画像認識タスク
- モデルの評価

## 環境設定

必要なライブラリ：
- Python 3.8+
- PyTorch
- Matplotlib
- NumPy
